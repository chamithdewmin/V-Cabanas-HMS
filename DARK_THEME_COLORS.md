# Dark Theme Colors Only

## Background Colors
- `#0c0e14` - Page background (`--background`)
- `#13161e` - Card background (`--card`)
- `#0f1117` - Table header background (`--table-header-bg`)
- `#111419` - Table odd row (`--table-row-odd`)
- `#13161e` - Table even row (`--table-row-even`)
- `#1a1d27` - Table hover row (`--table-row-hover`)

## Border Colors
- `#1e2433` - Border/Divider (`--border`, `--table-divider`)
- `#2a3347` - Table border (`--table-border`)

## Text Colors
- `#ffffff` - Primary text (`--text-primary`, `--foreground`)

- `#8b9ab0` - Label text (`--text-label`)
- `#6b7a99` - Muted text (`--text-muted`)

## HSL to Hex Conversions (Dark Theme)
- `225 25% 6%` = `#0c0e14` (Background)
- `224 22% 10%` = `#13161e` (Card)
- `217 91% 60%` = `#3b82f6` (Primary Blue)
- `217 55% 22%` = `#2a3a4f` (Sidebar Active BG)
- `210 40% 96%` = `#f0f4f8` (Foreground)
- `210 25% 18%` = `#2a3340` (Secondary)
- `210 30% 80%` = `#c4d0e0` (Secondary Foreground)
- `216 19% 15%` = `#1e2433` (Border)
- `0 72% 51%` = `#ef4444` (Destructive)
- `222 84% 56%` = `#5b8cff` (Ring)
